# DeepVariance Frontend

Machine learning platform for model development and dataset management, built with React + Vite + TypeScript.

## Features

- Dataset Management
- Model Registry
- Experiment Tracking
- Job Monitoring
- Dark/Light Theme

## Tech Stack

- React 18 + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- React Query + Zustand
- React Router v7

## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

```bash
npm install
cp .env.example .env
npm run dev
```

Configure `.env` with your API endpoint:
```env
VITE_API_BASE_URL=https://api.deepvariance.ai/v1
```

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run lint` - Run ESLint
